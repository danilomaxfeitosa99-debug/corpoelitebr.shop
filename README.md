# Solaris Energia - Versão CNPJ 65.009.634/0001-50

Esta pasta contém uma variação do site institucional da **Solaris Energia**, configurada para uso com o CNPJ **65.009.634/0001-50** e com textos ajustados em relação à versão base.

## Visão Geral

O site apresenta as soluções de energia solar para residências, empresas e propriedades rurais, reforçando os benefícios financeiros e ambientais da geração fotovoltaica própria.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Responsivo, leve e com foco em sustentabilidade.
-   **Objetivo:** Servir como landing page para campanhas e captação de leads qualificados.

## Funcionalidades

Esta variação mantém as mesmas interações da pasta `solaris-energia-static`:

1.  **Header dinâmico ao rolar a página;**
